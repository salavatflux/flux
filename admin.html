OBJECTSНажмите на серый прямоугольник чтобы загрузить фото. Изображение показывается целиком, не обрезается. На мобильном — два пальца для увеличения. Ориентация задаёт пропорции рамки.
PAPER SERIESКаждая серия — обложка + несколько работ внутри. На сайте: обложка серии видна в списке, по кнопке «к серии» открываются все работы.
DIGITALКаждая серия — отдельный слайд, листается стрелками. Внутри — лента из текстовых блоков и фотографий в любом порядке. Можно задать фоновое изображение (полупрозрачное) и кнопку-ссылку внизу.
STATEMENTКаждое поле = один абзац авторского текста.
CV / ВЫСТАВКИНажмите на ячейку чтобы загрузить фото. Нажмите на S / M / L чтобы изменить размер. × — удалить. + — добавить ячейку.
